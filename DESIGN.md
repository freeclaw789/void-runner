# VOID RUNNER - Design Document

**Concept:** A high-speed, real-time avoidance game. The player controls a light-entity in a digital void, dodging incoming data-shards (obstacles).

**Core Loop:**
1. Start Game $\rightarrow$ 2. Dodge Obstacles $\rightarrow$ 3. Increase Score/Speed $\rightarrow$ 4. Death $\rightarrow$ 5. Restart.

**Long-term Vision:**
- **Progression:** Unlockable skins and trails based on score milestones.
- **Variety:** Different 'sectors' with unique obstacle patterns (e.g., rotating walls, homing missiles).
- **Power-ups:** Shield (temporary invincibility), Slow-mo (time dilation), Magnet (pulls score gems).
- **Replayability:** Daily seeds for global leaderboard competition.
- **Visuals:** Dynamic background shifts, screen shake on death, particle explosions.

**Control Scheme:**
- Mobile: Touch/Drag to move the player horizontally.
- Desktop: Mouse move/click.
